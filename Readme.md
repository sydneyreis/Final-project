# Summary of user feedback

Overall reception was positive (avg rating 4/5), with users finding outfit suggestions well-matched to weather and dressiness preferences.

Common issues:

- Temperature/warmth scale was confusing or reversed
- Adding clothing to the system felt tedious and time-consuming
- Some input prompts were unclear (e.g. dress length, neutral temperature values)

Top requested improvements:

- Clearer scale labels with examples (e.g. actual temperature ranges)
- A "back" button during input
